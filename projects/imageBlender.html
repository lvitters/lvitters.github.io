{

image blender

2017 · p5.js, Processing
}

Inspired by Kim Asendorf's pixel sorting algorithm. This application allows users to place two images on a canvas and blend them together by sorting their pixels by hue values. The images are resized to the same height and sampled down for better performance. This project was made as a final project for a Creative Coding class at New York University.
