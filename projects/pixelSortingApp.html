{

pixel sorting app

2018 · android application
}

This android app allows users to do pixel sorting by hand, in realtime. Users can can apply sorting algorithms like sorting by hue or by saturation to differently shaped and sized selections of an image taken by their phone's camera. The image always consists of the same pixels, they are merely rearranged to create the effect of a pixel sorting "brush". Good for obscuring your face.
